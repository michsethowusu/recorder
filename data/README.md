# Data Folder

Place your language-specific CSV files here using the naming convention:

    data_{Language}.csv

Examples matching the LANGUAGES list in recorder.py:
    data_Twi.csv
    data_Dagbani.csv
    data_Ewe.csv

Each CSV must have at minimum a `text` column (and optionally an `id` column).
The original data.csv is provided as a reference/template.
